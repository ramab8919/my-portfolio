Case Study

Sensic

Designing & creating an accessible music learning experience for the Deaf community.

My Role

End-to-End Product Design

This was a team project, and I independently led the UX research and the end-to-end design process. I was solely responsible for understanding user needs, identifying pain points and product opportunities, designing the complete design system from scratch, creating every user interface, and developing the application using SwiftUI.

UX Research

Conducted UX research to understand how Deaf users learn, identify challenges, and uncover opportunities that led to integrating declarative and tactile memory into the learning experience.

Opportunity Mapping

Translated research findings into product opportunities and meaningful user experiences.

Learning Strategy

Applied declarative memory and tactile memory principles to strengthen learning through visual guidance and haptic feedback.

Design System

Designed the complete design system including colors, typography, spacing, components, accessibility rules and interaction states.

UI / UX Design

Designed every screen, interaction, prototype and user flow following Apple's Human Interface Guidelines.

Development

Developed the application using SwiftUI with the MVVM architecture.

The Challenge

Understanding the Problem

Learning music can be particularly challenging for Deaf and hard-of-hearing users because most educational experiences rely heavily on audio feedback. Existing solutions often overlook accessible learning methods and provide limited visual guidance, making it difficult for users to understand pronunciation and confidently practice on their own.

01

User Challenges

Conducted UX research to understand how Deaf users experience learning pronunciation and music, uncovering the main barriers that limited engagement and long-term learning.

02

Opportunity Discovery

Through research, I identified opportunities to improve learning by combining visual guidance, haptic feedback, and interactive practice into one accessible experience.

03

Memory-Based Learning

I explored the concepts of declarative memory and tactile memory, then transformed these findings into learning features that help users retain information through repetition, interaction, and touch.

04

From Research to Design

Every research insight was translated into the complete design system, user interface, and interaction patterns, ensuring that every screen solved a real user problem.
